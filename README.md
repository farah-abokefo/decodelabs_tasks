# 🚀 Project 1 – Rule-Based AI Chatbot

## Description

A deterministic chatbot built entirely with Python using rule-based logic. The chatbot responds to predefined user inputs, demonstrating how AI systems can make decisions without machine learning.

### Features

- Dictionary-based knowledge base
- O(1) response lookup
- Keyword matching
- Conversation history (JSON)
- Session statistics
- Input sanitization
- Multiple exit commands
- Friendly chatbot personality

### Technologies

- Python
- JSON
- Regular Expressions (Regex)

---

# 🌸 Project 2 – Iris Flower Classification

## Description

A machine learning classifier built using the famous Iris dataset.

The project demonstrates supervised learning using the **K-Nearest Neighbors (KNN)** algorithm.

### Features

- Data preprocessing
- Feature scaling
- Train/Test split
- KNN classification
- Cross-validation
- Hyperparameter tuning
- Confusion Matrix
- Classification Report
- Accuracy evaluation

### Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

### Dataset

- Iris Dataset
- 150 flower samples
- 3 flower species
- 4 numerical features

**Model Accuracy:** ~98%

# 🎯 Project 3 – AI Recommendation System

## Description

A content-based recommendation engine that recommends technology careers based on user skills.

The system uses **TF-IDF Vectorization** and **Cosine Similarity** to calculate the similarity between user skills and job requirements.

### Features

- Skill-based recommendations
- TF-IDF Vectorization
- Cosine Similarity
- Top 3 job recommendations
- Confidence scores
- Cold-start handling

### Technologies

- Python
- Pandas
- Scikit-Learn
