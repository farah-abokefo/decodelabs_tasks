## 📋 Overview

This is **Project 1** of the Artificial Intelligence Engineering track at **DecodeLabs**. It demonstrates the fundamental concepts of deterministic AI through a **Rule-Based Chatbot** that operates entirely on explicit logic and control flow.

> *"Before you can manage the chaos of a probability engine, you must master the precision of a logic engine."*

### 🎯 Key Concepts

- **IPO Model**: Input → Process → Output architecture
- **Deterministic Logic**: Predictable, traceable responses
- **White Box Design**: Fully transparent decision-making
- **Hash Map Lookup**: O(1) efficient response retrieval
- **Control Flow**: Continuous loop with exit strategies

---

## ✨ Features

### Core Features
- ✅ **Rule-Based Responses** - Predefined responses using dictionary mapping
- ✅ **Continuous Loop** - Stays active until exit command
- ✅ **Input Sanitization** - Handles case, whitespace, and special characters
- ✅ **Hash Map Lookup** - O(1) response retrieval with fallback
- ✅ **Keyword Pattern Matching** - Handles variations of similar queries
- ✅ **Clean Exit Strategy** - Multiple exit commands (bye, exit, quit, etc.)

### Enhanced Features
- 🎭 **Personality System** - Catchphrases, emojis, and encouraging responses
- 📝 **Chat History** - Automatic saving and loading of conversations
- 📊 **Session Statistics** - Track exchanges, duration, and usage patterns
- 💾 **Export Functionality** - Save conversations as JSON files
- 🔍 **History Viewing** - Review past conversations
